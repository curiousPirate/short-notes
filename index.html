CSS

  • CSS is the language we use to style an HTML document. CSS describes how HTML elements should be displayed.
  • CSS stands for Cascading Style Sheets. CSS describes how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work. It can control the layout of multiple web pages all at once. External stylesheets are stored in CSS files.

GIT

  • git status = check on which branch we are working
  • git add -A = Stage the changes
  • git commit -m "short msg about changes" = commit changes
  • git pull origin main = to check changes are updated before pull request
  • git push origin feature/CSS = to push the feature/CSS branch changes to the repo
  • git checkout main = switches from one branch to another
  • git pull origin main = to pull the code from the repo
  • git checkout -b branch-name = creates a new branch and switches to it
  • git branch = to check available branches
  • git branch -r = to check remote branches
  • git branch -a = for all branches
  • git branch -d = to delete specific branches, -D to force delete

JS

  • A variable in a javascript is a container for strings, integers etc;.. the data can be changed as required. “var” is the keyword used to declare variable.
  • console.log is a method to output the value. console.log(topics), in this argument is placed inside the paranthesis and output would be the data assigned in the “topics” variable.
  • What exactly do we mean by “truthy” and “falsy” in Javascript? A value is considered **falsy** if it is one of the following values: `0`, `-0`, `On`, `""`, `null`, `undefined`, `NaN`, or a **Boolean** data type with the value of `false`. All other values, including a Boolean with the value of `true`, are considered **truthy**.
  • Use "if" to specify a block of code to be executed, if a specified condition is true.
  • Use "else" to specify a block of code to be executed, if the same condition is false.
  • Use "else if" to specify a new condition to test, if the first condition is false.
  • if (condition) { // block of code to be executed if the condition is true }
  • Use "for" loop if we want to repeat a piece of code.
  • for (let i = 0; i < cars.length; i++) { console.log(cars); }
  • A JavaScript function is a block of code designed to perform a particular task. It is executed when we call it.
  • Function to compute the product of p1 and p2, "function myFunction(p1, p2) { return p1 * p2; }"
